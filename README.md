# Gestor de Agenda de Contactos

Este proyecto es un sistema de gestión de contactos que permite a los usuarios almacenar, organizar y buscar información de sus contactos de manera eficiente. Tiene el objetivo de poner en practica fundamentos de programación con C++ al manejar estructuras de datos como ArrayList, listas enlazadas, pilas, colas y árboles binarios de búsqueda.

## Requisitos Técnicos

- Lenguaje de programación: C++.
- Visual Studio 2019 o superior.

## Instrucciones de Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/Mari250607/GestorAgendaContactos.git
   ```
2. Abre el proyecto en Visual Studio:

   - Navega a la carpeta donde clonaste el repositorio.
   - Abre el archivo .vcxproj con Visual Studio.

3. Compila el proyecto:
   - En Visual Studio, selecciona "Build" > "Build Solution" o presiona `Ctrl + Shift + B`.
4. Ejecuta el programa:
   - Después de compilar, selecciona "Debug" > "Start Without Debugging" o presiona `Ctrl + F5`.
